[`CeasorCodes`](https://ceasorcodes.vercel.app/) is a blog | portfolio website created by[`CeasorCodes`](https://ceasorcodes.vercel.app/)

This Next.js project is a modern web application that leverages the power of TailwindCSS and Shadcn for a sleek, responsive design.
It utilizes MDX to seamlessly integrate JSX within markdown content, allowing for dynamic components like interactive charts and alerts to be embedded directly into the narrative flow.
For an enhanced developer experience, it features a Rehype plugin with Shiki syntax highlighting, ensuring that code blocks are not only visually appealing but also accurately represented, both during server-side builds and on the client-side for real-time interactions.

## Description

The blog | portfolio is developed using [`rehype-pretty-code`](https://github.com/atomiks/rehype-pretty-code) is a Rehype plugin powered by the
[`shiki`](https://github.com/shikijs/shiki) syntax highlighter that provides beautiful code blocks for Markdown or MDX. It works on both the server at build-time (avoiding runtime syntax highlighting) and on the client for dynamic highlighting.
